## Description

Este é um projeto simples para aprendizado do framework NestJs. Neste projeto vou abordar um CRUD do domínio de cursos. A seguir instruções de instalação, e ferramentas utilizadas.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
